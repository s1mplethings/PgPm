## 如何得到与截图一致的界面并测试规则
1. 运行 `pnpm tauri dev` 启动桌面端。
2. 在首页点击 **“测试工具 → 导入示例数据（v0.4 截图版）”**  
   - 预期：左侧推荐卡显示 **T-001**，下方五列出现 **T-100 ~ T-141**。
3. 点击 **“制造 ≤24h 到期（T-110）”**  
   - 预期：推荐卡切换到 **T-110**，因为 24h 内任务必须进入重点。
4. 点击 **“模拟 WIP 已满”**  
   - 预期：推荐区域出现 “WIP 上限已满 / 无可执行任务”。
5. 点击 **“打开数据目录”**  
   - 预期：打开的目录包含 `tasks.json / settings.json / events.json / backups/`，可直接查看持久化数据。
6. 点击 **“清空数据”** 可回到空状态，如需恢复再导入 v0.4 示例。
